# sensor_-library
## 传感器库函数编写计划

#### 1. 实验目的

	*  提高代码复用性，为以后的开发提供帮助
	*  借此机会熟悉芯片工作流程以及常见通讯方法
	*  提高相关开发水平



#### 2. 实验要求 

* 为现有每一个传感器编写一套针对stm32 、esp32、树莓派的库函数并且能够正常运行
* 考虑可移植性，为后面的工作提供条件
